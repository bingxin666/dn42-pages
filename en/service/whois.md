# Whois Server

::: warning
This service may be in testing. It can be unstable and may introduce breaking changes at any time.
:::

We provide a Whois server and allow queries from different IRR sources. Currently supported IRR sources:

| IRR Source | Server Address | Support Types |
| --- | --- | --- |
| Mixed | `whois.baka.dn42` | Whois |
| Mixed | `irr.baka.dn42` | IRR |
| [DN42 Registry](https://git.dn42.dev/dn42/registry) | `dn42.whois.baka.dn42` | Whois / IRR |
| [BPDB](https://git.dn42.dev/BPDB/BPDB) | `bpdb.whois.baka.dn42` | Whois / IRR |
| Mixed                                                      | `https://rdap.baka.dn42`                 | RDAP              |
| [DN42 Registry](https://git.dn42.dev/dn42/registry)       | `https://dn42.rdap.baka.dn42`          | RDAP      |
| [BPDB](https://git.dn42.dev/BPDB/BPDB)                    | `https://bpdb.rdap.baka.dn42`          | RDAP      |
