# Email Service

We provide free DN42 domain email addresses to all players with a valid DN42 ASN. The email suffix is `x.dn42`.

## Application Process

You need to apply via [@baka_lg_bot](https://t.me/baka_lg_bot) on Telegram:
 - Use the `/login` command to log in and verify your DN42 ASN.
 - Use the `/email` command to apply for an email address.

### Notes
 - Each ASN can apply for only one email address.
 - The email server is provided without an SLA guarantee.

## Email Server Information
 - Email server address: `mail.x.dn42`
 - Supports IMAP and SMTP protocols, as well as PLAIN and SSL/TLS encryption methods.
