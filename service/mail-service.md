# 邮件服务

我们向所有拥有有效 DN42 ASN 的玩家提供免费的 DN42 域名邮箱，邮箱后缀为 `x.dn42`。

## 申请流程

您需要在 Telegram 使用 [@baka_lg_bot](https://t.me/baka_lg_bot) 进行自助申请：
 - 使用 `/login` 命令登录，输入您的 DN42 ASN 进行验证。
 - 使用 `/email` 指令申请邮箱。

### 注意事项
 - 每个 ASN 只能申请一个邮箱。
 - 邮箱服务器无 SLA 保证。

## 邮箱服务器信息
 - 邮箱服务器地址: `mail.x.dn42`
 - 支持 IMAP 和 SMTP 协议，支持 PLAIN 和 SSL/TLS 加密方式。
